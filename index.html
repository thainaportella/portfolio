TEMPERATURE BLANKET

To create a crochet project that involves making one row per day throughout the year based on the daily temperature, I needed a way to record the temperature at a specific time of my choice. To achieve this, I decided to use technology to my advantage and built a scheduling system using Java and Spring Boot. The system records the temperature at the defined time and saves it in a CSV file along with the corresponding color. It also allows me to define the location from which the temperature should be retrieved each day. I continuously improve this project on a weekly basis, and the next step is to develop a front-end application that will allow me to download an app on my phone, making it easier to access and track the records.
